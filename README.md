# The Ultimate Python Bootcamp

This repository contains my work from the Udemy course **The Ultimate Python Bootcamp: Learn by Building 50 Projects**. It follows a project-based learning path across core Python, data handling, web scraping, automation, concurrency, async programming, data science, Flask, and Pydantic.

The goal of this repo is not just to collect exercises, but to show how my Python skills progressed from small utility scripts to more advanced, structured projects.

## What I Learned

- Writing Python scripts for daily tasks and reusable utilities
- Reading from and writing to text, CSV, and JSON files
- Working with APIs and structured data
- Scraping websites with `requests` and `BeautifulSoup`
- Automating file and folder workflows
- Exploring data with `pandas`, `numpy`, and notebooks
- Building web apps with Flask
- Using threading, multiprocessing, and async programming
- Validating and modeling data with Pydantic
